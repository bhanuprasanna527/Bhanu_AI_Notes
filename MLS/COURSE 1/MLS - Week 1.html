Overview of Machine Learning Last updated: 2019-06-01

Machine Learning Definition:

Machine learning is a branch of artificial intelligence (AI) and computer science which focuses on the use of data and algorithms to imitate the way that humans learn, gradually improving its accuracy.

Welcome to Machine Learning

Machine learning has been a growing exponentially year on year. Many companies like Apple, Facebook, Google use extensive Machine Learning to create new technologies that are make lives better. Machine Learning Reasearch has always been active where researchers are developing new models everyday and testing them how are they performing compared to other models.

Supervised vs Unsupervised Machine Learning

This section deals with the differences between Supervised and Unsupervised Machine Learning.

What is Machine Learning?

Machine learning definition: Field of Study that gives computers the ability to learn without being explicity programmed.

The most used learning algorithms are supervised learning and recommender learning. Let's start learning Machine Learing.

Supervised Learning

Machine Learning is creating tremondous value.

Supervised learning is when the data is labelled i.e, we have both input and output values. The supervised algorithm trains to fit the inputs with outputs to gain accuracy. So, finally Supervised learning is giving X inputs and taking Y outputs.

Types of Supervised Learning

Supervised learning can be further divided into two types of problems:

Regression

Regression algorithms are used if there is a relationship between the input variable and the output variable. It is used for the prediction of continuous variables, such as Weather forecasting, Market Trends, etc. Below are some popular Regression algorithms which come under supervised learning:

  1. Linear Regression
  2. Regression Trees
  3. Non-Linear Regression
  4. Bayesian Linear Regression
  5. Polynomial Regression

Classification

Classification algorithms are used when the output variable is categorical, which means there are two classes such as Yes-No, Male-Female, True-false, etc.

  1. Random Forest
  2. Decision Trees
  3. Logistic Regression
  4. Support Vector Machines

To summarize Regression: Regression is used to predict a number infinitely many possible outputs.

To summarize Classification: Classification predicts categories of small number of possible outputs.

Unsupervised Learning

Unsupervised learning is when there is no labelling of data to get the output.

Unsupervised learning is used to get the patterns and structures from the given output.

Unsupervised learning cannot apply a regression or classification problem because unlike supervised learning, we have the input data but no corresponding output data. The goal of unsupervised learning is to find the underlying structure of dataset, group that data according to similarities, and represent that dataset in a compressed format.

The unsupervised learning algorithm can be further categorized into two types of problems:

Unsupervised algorithm has to find structure in the data

Types of Unsupervised learning:

  1. K-means Clustering
  2. KNN (K-Nearest Neighbour
  3. Dimensionality Reduction
  4. Hierarchal Clustering
  5. Anamoly Detection
  6. Neural Networks
  7. Principle Component Analysis
  8. Independent Component Analysis
  9. Apriori Algorithm
  10. Singular value decomposition

Clustering

Clustering algorithm is an unsupervised algorithm. Clustering is a process of seperating different patterns that can be grouped into a specific set of points which can be visualized as a cluster.
