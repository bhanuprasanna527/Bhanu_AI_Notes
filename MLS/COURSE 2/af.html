Binary Activation Function¶

  • The Binary Activation function is demonstrated as:
$$f(x) = \left\{ \begin{array}{lll} 0 & for & x \leq 0 \\ 1 & for & x > 0 \end{array} \right.$$
In [4]:
def binary_activation_function(x):
    return 0 if x < 0 else 1
In [5]:
y_bin = [binary_activation_function(i) for i in x]
In [6]:
plt.plot(x, y_bin)
plt.grid(True)
plt.title("Demonstration of Binary Activation Function")

ReLU Activation Function¶

  • The ReLU (REctified Linear Unit) Activation Function is demonstrated as:
$$f(x) = \left\{ \begin{array}{lll} 0 & for & x \leq 0 \\ x & for & x > 0 \end{array} \right.$$
In [16]:
def relu_activation_function(x):
    return x if x > 0 else 0
In [17]:

In [18]:

plt.grid(True)
plt.title("Demonstration of ReLU Activation Function")

Leaky ReLU Activation Function¶

  • The Leaky ReLU (REctified Linear Unit) Activation Function is Demonstrated as:
$$f(x) = \left\{ \begin{array}{lll} ax & for & x \leq 0 \\ x & for & x > 0 \end{array} \right.$$
In [19]:
def leaky_relu_activation_function(a, x):
    return a * x if x <= 0 else x
In [20]:
y_lrelu = [leaky_relu_activation_function(0.1, i) for i in x]
In [21]:
plt.plot(x,y_lrelu)
plt.grid(True)
