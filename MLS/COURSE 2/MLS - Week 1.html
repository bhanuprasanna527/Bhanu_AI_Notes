    Linear Regression Layer using Tensorflow

    Linear Regression can be implemented in Neural Network also. Here, the activation function used will be Linear.

    Linear Regression Function:

    $$ f_{\mathbf{w},b}(x^{(i)}) = \mathbf{w}\cdot x^{(i)} + b \tag{1}$$

    
import tensorflow as tf
linear_model = Sequential([ tf.keras.layers.Dense(1, input_dim=1,  activation = 'linear', name='Linear') ])

    This is the layer of Linear Regression in Tensorflow using Keras.

    Logistic Regression Layer using Tensorflow

    The function implemented by a neuron/unit with a sigmoid activation creates a Logistic Regression.

    Logistic Regression Function:

    $$ f_{\mathbf{w},b}(x^{(i)}) = g(\mathbf{w}x^{(i)} + b) \tag{2}$$

    where$$g(x) = sigmoid(x)$$

    
import tensorflow as tf
logistic_model = Sequential([ tf.keras.layers.Dense(1, input_dim=1,  activation = 'sigmoid', name='Logistic') ])

    This is the layer of Logistic Regression in Tensorflow using Keras.

    Forward Propagation

    Here we will discuss about Forward Propagation in Neural Network. Let us take ANN (Artificial Neural Network as the model for Forward Propagation.

    Artificial Neural Network

  • Artificial neural networks are computing systems inspired by the biological neural networks that constitute animal brains.
  • An ANN is based on a collection of connected units or nodes called artificial neurons, which loosely model the neurons in a biological brain.


  • Neurons are arranged in layers and layers are arranged sequentially.
  • Neurons within the same layer do not transfer information with each other.
  • All data/info enter via the input layer and information/output goes out by the output layer.
  • All Neurons in layer ‚Äú l ‚Äù are connected to all neurons in layer ‚Äú l+1 ‚Äù.
  • Every interconnection in the neural network has a weight associated with it, and every neuron has a bias associated with it.
  • All Neurons in a particular layer use the same activation function.

    • Steps for Forward Propagation

  • Calculating, Z =summation[(weights*input)+bias].
  • Choosing Activation function = for binary classification sigmoid function.
  • Substituting the value of ‚Äú Z ‚Äù, we will get y_pred.
  • Calculation of Loss using binary cross-entropy.
